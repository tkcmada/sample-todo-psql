## Acceptance Criteria

Run `npm run ci` before committing. This executes:

- `npm run fmt` (Prettier formatting check)
- `npm run typecheck` (TypeScript type check)
- `npm run lint` (ESLint)
- `npm test -- --run` (Vitest)
